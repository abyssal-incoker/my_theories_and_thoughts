# 数学理论： 喜欢流分析法
这个理论的核心是基于化归核心思想的，以特征识别为核心的迭代式启发性知识搜索策略，这里暂时只用不定积分举例，这个理论的拓展应用远不止不定积分，研究好需要运用到的知识的性质，就可以通过这个理论来帮助训练思维，想到好方法，值得一提的是，这种方式规则较为明确，如果以后可以让ai执行，那么我们可能可以获得启发性检索的，算力要求低且高效的透明白箱ai。

不定积分决策——喜欢流🥰🥰🥰分析法🧐
1.核心释义——什么是喜欢🥰﻿
其实数学是一个巨大的嘎啦给木（galgame）！﻿
喜欢流分析法，顾名思义，是谁在喜欢，是喜欢什么呢？其实很简单，是基本积分表，代数变形公式在喜欢，喜欢式子中出现的她们含有的对应形式。这种形式可以是直接具体的对应，也可以是抽象的特质对应（比如三角函数喜欢平方，基本积分公式表分母那里喜欢二次，或者反正切型的根号嵌套平方）这种喜欢其实是对运算性质与条件的充分理解与应用﻿
2.数学与信息理论的核心思想😃﻿
这种思想的核心是化归特殊，将一切复杂问题化归为特殊情况的组合与嵌套。这个决策方向为什么好，因为它最充分利用了已知条件与知识，从陌生一般情况，逐渐通过“喜欢”的特质指引，一步步靠近特殊情况的化归。这在决策中有效提供了规律指引与反馈信息，大大减少了决策时的信息熵，从而减少了决策成本。﻿
3.具体决策流程🤓﻿
①先扫描式子中的元素类型，代数结构，回想对应元素与对应结构可用的、较为直观简便的变形公式（恒等式，展开，凑微分，换元，对称结构特殊变形之类的）﻿
②分析元素类型，代数结构，基本积分表她们喜欢什么。这个甚至可以通过基本公式的底层研究，从元素和形式上归纳出喜欢的情况，作为基本知识储备﻿
③从可选用的方法中挑选出“喜欢”决策路线，使用“喜欢”形式指引的方向进行变形﻿
通常就可以得到成本很低的决策路线。如果失败，可能需要考虑对基本公式与运算的熟悉程度，归纳记忆程度，还有现有知识储备问题。﻿
值得一提的是，在挑选“喜欢”决策时，不同视角会诞生不同方法。﻿
④这个决策的很大特点就是每一个“喜欢”指引，也会和“喜欢”变形后的形式产生进一步的对应，这就形成了，先由于“喜欢”特征指引产生初步变形，一步步迭代，每一步都有一个“喜欢”思维接口，这样就解决了无法一次性预判复杂代数情况，一下想到变形的困难。😋﻿
4.意义升华🤔﻿
其实这个理论看似是不定积分理论，但实际上不定积分提到得比较少，这个理论的抽象性和普适性可能远远超出不定积分决策的范畴，直指代数变形的本质（我看可能更广，不只数学领域，人类解决所有问题几乎都要化归特殊）。﻿
这个理论为人类思维的黑箱提供了一个解剖图，可以供初学者快速学习，建立强大的直觉，也可以写成算法让ai学习，打破传统ai数据依赖与算力限制还有黑箱不可解释性？😳﻿
